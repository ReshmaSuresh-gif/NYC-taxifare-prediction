# New York City Taxi Fare Prediction

# Overview
This project focuses on predicting taxi fares in New York City using machine learning techniques. The dataset contains historical taxi trip data, which is used to analyze patterns and build predictive models.

# Objective
To develop machine learning models that can accurately predict taxi fares based on trip details such as distance, duration, and number of passengers.

# Dataset
- Source: Kaggle  
- Dataset: New York City Taxi Fare Prediction  
- Link: https://www.kaggle.com/datasets/dansbecker/new-york-city-taxi-fare-prediction/data  

# Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

# Features Used
- Trip duration  
- Distance traveled  
- Number of passengers  
- Fare  
- Tip  
- Miscellaneous fees  
- Surge applied  

# Project Workflow
- Data Loading  
- Data Cleaning  
- Exploratory Data Analysis (EDA)  
- Feature Selection  
- Model Building  
- Model Evaluation  

# Machine Learning Models Used
- Logistic Regression  
- Decision Tree Classifier  
- K-Nearest Neighbors (KNN)  
- Random Forest  
- Support Vector Classifier (SVC)  
- Naive Bayes  

# Results
The models were trained and evaluated to compare their performance in predicting taxi fares. Random Forest and Decision Tree models showed better performance compared to others.
